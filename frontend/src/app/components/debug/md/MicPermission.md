# MicPermission

## 功能描述

麦克风权限状态显示组件。

## 状态值

| 状态 | 显示文本 | 含义 |
|------|----------|------|
| `granted` | "mic: ✓" | 麦克风已授权 |
| `denied` | "mic: ✗" | 麦克风被拒绝 |
| `prompt` | "mic: ?" | 等待用户授权 |

## 核心方法

| 方法 | 描述 |
|------|------|
| `setPermission(permission: string)` | 更新权限状态 |

## 使用场景

- 检查麦克风是否可用
- 调试录音功能问题
